# PERSONAL EXPENSE TRACKER


# OBJECTIVE
Budget Buddy is a simple, Python based application which helps people track their monthly set budget , its pretty fast and easy to use considering that it saves all the provided data in between the sessions.


# FEATURES
IT HELPS SET MONTHLY BUDGET AT THE START OF THE MONTH
IT RECORDS EXPENSES INCLUDING THE AMOUNT AND THE CATEGORY, ONE CAN EVEN ADD NEW EXPENSES
ALL THE DATA PROVIDED IS SAVED TO A LOCAL FILE
ONE CAN REVIEW A NUMBERED LIST OF ALL RECORDED EXPENSES
THERES'S REAL TIME TRACKING
ISSUES SMART WARNINGS TO KEEP USER IN CHECK

# TOOLS USED
Language: Python
Editor: Python IDLE
References: Financial data from different websites


# STEPS TO INSTALL AND RUN THE PROJECT

Install Python 
Open a new file in IDLE Shell
Create a file name personal_expense_tracker.py
Write code and save it
Execute code
Enter set your Total budget this month and get output


# INSTRUCTIONS FOR TESTING
Start the program
Add expense :Enter the amount and a brief description like rent
The total spent is updated and saved immediately
View expenses :Lists all the expenses you have recorded so far
Check remaining :It shows the full given budget summary and the calculated safe daily amount
Quit :Exits the application ensuring the data is saved
